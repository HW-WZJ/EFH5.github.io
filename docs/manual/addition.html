索引

附加内容：

page_nav.css

HTML代码：


<nav class="page-nav">
	<input type="checkbox" id="nav-chb">
	<label for="nav-chb" class="page-nav-lable">菜单</label>
	<div id="nav-box">
		<ul>
			<li class="page-nav-li"><a href="#section0" id="pnav0" class="page-nav-a page-nav-at">页首</a></li>
			<li class="page-nav-li"><a href="#section1" id="pnav1" class="page-nav-a">第一部分标题</a></li>
			<li class="page-nav-li"><a href="#section2" id="pnav2" class="page-nav-a">第二部分标题</a></li>
			<！-- 以此类推，可以多添加几个 -->
			<li class="page-nav-li"><a href="#" class="page-nav-a">回到顶部</a></li>
		</ul>
	</div>
	<script>
		$(function() {
			$(window).scroll(function() {
				//为页面添加页面滚动监听事件
				var wst = $(window).scrollTop() //滚动条距离顶端值
				console.log("滚动条距离顶端值:" + wst)
				var len = 3;//这里的数字根据需要替换，有n个部分就写上n+1，如2部分就写3
				for (i = 0; i < len; i++) { //加循环
					console.log("section" + i + "距离顶端值:" + $("#section" + i).offset().top);
					var next = i + 1;
					var itemOffsetTop = $("#section" + i).offset().top;
					console.log(i);
					if (i === (len - 1)) {
						var condition = itemOffsetTop <= wst;
					} else {
						var itemNextOffsetTop = $("#section" + next).offset().top;
						var condition = itemOffsetTop <= wst && itemNextOffsetTop >= wst;
					}
					if (condition) { //判断滚动条位置
						$('a').removeClass("page-nav-at"); //清除c类
						$("#pnav" + i).addClass("page-nav-at"); //给当前导航加c类
					}
				}
			})
			$('#nav a').click(function() {
				$('#nav a').removeClass("c");
				$(this).addClass("c");
			});
		});
	</script>

以EFH5的侧边栏为例，HTML代码如下：


<div class="main-sidebox">
	<div class="main-sidebox-section">
		<div class="main-sidebox-section-h">EFH5</div>
		<div class="main-sidebox-siteicon-pic"></div>

	</div>
	<div class="main-sidebox-section">
		<div class="main-sidebox-section-h"><i class="fas fa-link"></i>友情链接</div>
		<script>sidebox_links();</script><noscript>这里需要使用 JavaScript 进行加载！</noscript>
	</div>
</div>

这里还使用了JavaScript以做到可以轻松更换友情链接。

信息盒子

附加内容：

infobox.css
Font Awesome

推荐位置：section下一级

HTML代码示例：


<div class="ib ib-yellow"><div class="ib-fi ib-yellow-h"><i class="fas fa-info-circle"></i></div>
	<div class="ib-la"><big><b>信息盒子标题</b></big><br>内容</div>
</div>

<font class="ib-purple-d-hl">突出显示</font>

主题选择

依赖：

setting boxsetting_box.css
主题颜色theme_color.css

附加内容：

theme_select.css
theme_select.js

推荐位置：section同级

HTML代码示例：


<div class="setting-box">
	<input type="checkbox" id="darkmode-chb" class="setting-chb">
	<label for="darkmode-chb" class="darkmode-chb-lable setting-chb-lable"></label>
	<input type="checkbox" id="greenmode-chb" class="setting-chb">
	<label for="greenmode-chb" class="greenmode-chb-lable setting-chb-lable"></label>
</div>

此处给出的主题有三种，分别为默认的colorfulmode、深色主题darkmode和绿色主题greenmode。主题的相关配色都在theme_select.css里面进行设置。当然，你也可以为你的EFH5增添更多主题配色。

各个按钮的状态以及主题的切换都是由theme_select.js控制的，所以要添加主题配色或者更改其名称需要一并调整theme_select.js。

另外，若要实现自动切换深色主题（部分支持），需要将theme_select.css中/* Basic Styles */部分放在最先加载的部分（即regular.css的最前；或者在HTML中作为第一个载入的stylesheet）。
