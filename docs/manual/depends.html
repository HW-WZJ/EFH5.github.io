依赖CSS

这部分CSS所控制的样式会多次反复使用。

setting box

附加内容：

setting_box.css

与section同级。

用于提供一些简单的设置区域，一般建议2-4个选择框。

例子：插件-主题选择

page button box

附加内容：

page_button_box.css

与section同级。

用于上下翻页等有两个超链接按钮的情况。也可以将其配色设置为与setting box相同。不过那样setting box就又是这个的依赖了。

例子：插件-翻页

美化

这部分CSS所控制的样式主要用于页面美化。

主题颜色

附加内容：

theme_color.css

用于提供一些配色的控制。

下面是theme_color.css中变量所代表的意义(可根据需要自行增减)：

--text-color 默认文本颜色
--bgcolor 页面背景颜色
--section-bgcolor section的背景颜色
--normal-header-bgcolor 页首背景颜色
--flex-header-bgcolor 导航栏（上）背景颜色
--flex-header-shadow-color 导航栏（上）下方光晕颜色
--main-sidebox-section-text-color 侧边栏（右）默认文本颜色
--main-sidebox-section-text-a-color 侧边栏（右）超链接颜色
--setting-box-bgcolor setting box背景颜色
--table-bgcolor1 表格美化背景颜色1
--table-bgcolor2 表格美化背景颜色2

代码显示

附加内容：

code.css

用于提供代码块（<pre>与<code>）的样式。

表格美化

附加内容：

sheet.css

用于提供表格的样式，包括表格标题（表头）、奇偶行区分等。

动效

这部分CSS所控制部分动画效果。

neon

附加内容：

neon.css

提供了霓虹灯开启时的闪几下的动效。
